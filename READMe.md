# Calculadora Web — Proyecto JS

Este proyecto es una **calculadora web interactiva** desarrollada con **HTML, CSS y JavaScript puro**.  
Forma parte de un ejercicio práctico para aprender manipulación del DOM, eventos, funciones y estructura modular de código en JavaScript.

---

## Características principales

- **Operaciones básicas:** Suma, Resta, Multiplicación y División.  
- **Operaciones avanzadas:** Potenciación (`Math.pow`) y Raíz n-ésima con validaciones para índices y números negativos.  
- **Funciones adicionales:**  
  - Redondeo normal, hacia arriba y hacia abajo.  
  - Opción para mostrar el resultado original con decimales.  
  - Botón para limpiar los campos.  
  - Animación visual al actualizar el resultado.  
- **Diseño visual:** Interfaz moderna, limpia y adaptable, con efectos de hover y transiciones suaves.  

---

## Tecnologías utilizadas

- **HTML5:** estructura semántica del contenido.  
- **CSS3:** estilos, colores, animaciones y diseño responsive.  
- **JavaScript (Vanilla JS):** lógica funcional, operaciones y manipulación del DOM.

---

## Funcionalidad destacada

El sistema guarda de forma temporal el **último resultado** para permitir aplicar operaciones adicionales (como redondeos) sin recalcular.  
También cuenta con una animación de resaltado (`.activo`) que se activa cada vez que el resultado cambia.
